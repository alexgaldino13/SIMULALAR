# Tarefa 1: Planejamento de Hosting e Nuvem (Dante Deploy)

**Objetivo:** Definir e preparar o ambiente de produção para o backend Django.

**Instruções:**
1. Analise o projeto Django em `simulacao/` e identifique dependências.
2. Prepare um arquivo de recomendações para o Railway.app ou Render.com.
3. Crie um `Dockerfile` básico para garantir que o app rode em qualquer container.
4. Identifique as variáveis de ambiente necessárias (DATABASE_URL, SECRET_KEY, DEBUG, etc).

**Saída esperada:** Relatório com guia de deploy e arquivos de configuração básicos.
