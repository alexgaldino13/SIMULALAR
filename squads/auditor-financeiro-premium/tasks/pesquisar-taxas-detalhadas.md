# Tarefa: Pesquisar Taxas Detalhadas 🔍
**Agente:** Rita Referência

## 🎯 Objetivo
Localizar e formatar os dados financeiros reais de mercado para servirem de benchmark na auditoria.

## 📋 Instruções
1. Identifique as taxas vigentes para os bancos: **Caixa, Inter, Santander**.
2. Colete especificamente:
   - CET (Custo Efetivo Total) para SBPE (SAC e Price).
   - Tabela de Seguro MIP (Morte e Invalidez Permanente) por faixa etária (Principalmente: 30, 40, 50, 60 e 70 anos).
   - Coeficiente de Seguro DFI (Danos Físicos ao Imóvel) - Geralmente um % do valor do imóvel.
   - Taxa de Administração Mensal aproximada.
3. Para o segmento de **Aluguel**, busque no QuintoAndar:
   - Taxa de administração média cobrada do proprietário.
   - Yield (rendimento) residencial médio para capitais brasileiras (ex: SP/RJ).
4. Organize tudo no arquivo de saída em formato de tabela Markdown.
5. Liste as URLs das fontes e a data de consulta para cada dado.

## 📥 Input
- Arquivo: `research-focus.md` (Contém o foco da pesquisa do dia).

## 📤 Output
- Arquivo: `research-brief.md` (Tabelas e fontes consolidadas).
